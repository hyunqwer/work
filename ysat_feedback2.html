학력평가 리포트 성적추이 개선안

꺾은선 그래프 하나만 사용해야 한다면,
최근 4회까지만 보여주는 것이 가장 적절합니다

그래프 라이브러리 제약상 꺾은선 차트만 사용할 경우, 차트 종류를 바꾸기보다 데이터 개수별 배치 방식과 해석 문구를 조정해야 합니다. PDF 리포트의 제한된 카드 영역에서는 최근 3~4회가 가장 읽기 쉽고, 5회 이상은 가독성과 해석성이 떨어질 가능성이 큽니다.

1. 최종 권장안

PDF 리포트의 성적추이 영역은 최근 4회까지 표시하고, 5회 이상 이력이 있는 경우에는 최근 4회만 노출하는 방식이 가장 적합합니다.

권장 최대 표시 회차
4회
최근 2년 흐름 기준
6개월 주기와 잘 맞음 학력평가는 약 6개월마다 시행되므로 4회는 최근 2년의 흐름을 보여줍니다.
PDF 카드 영역에 적합 작은 카드 안에서 점수·등급·날짜 라벨을 함께 표시하려면 4회가 현실적인 상한입니다.
추이 해석에 충분 3회부터 추이가 보이고, 4회는 일시적 변동인지 꾸준한 변화인지 판단하는 데 도움이 됩니다.
5회 이상은 과밀 5회 이상 표시하면 점 간격이 좁아지고 라벨이 겹쳐 학부모가 핵심 변화를 읽기 어려워집니다.
최종 기준: PDF 리포트는 최근 4회까지만 표시하고, 전체 장기 이력은 웹 리포트 또는 상세 페이지에서 제공하는 구조를 권장합니다.

2. 응시 횟수별 표시 기준

응시 이력 표시 방식 그래프 데이터 처리 상단 해석 문구
1회 중앙 기준점 앞뒤에 빈 슬롯을 넣어 점 하나를 중앙에 배치 첫 평가 기준점
다음 평가부터 변화가 표시됩니다.
2회 직전 대비 비교 앞뒤에 빈 슬롯을 넣어 두 점이 양 끝에 붙지 않도록 배치 직전 대비 +○점 · 등급 유지/상승/하락
3회 최근 평가 흐름 최근 3회 표시, 좌우 여백 확보 직전 대비 +○점 · 최근 상승/하락/유지
4회 최근 2년 흐름 최근 4회 표시, PDF 리포트의 기본 최대값 최근 4회 흐름 · 직전 대비 변화
5회 이상 최근 4회만 표시 전체 데이터 중 최신 4개만 사용 최근 4회 기준 · 전체 이력은 웹에서 확인

4. 개발 적용 규칙

1

앞뒤 빈 슬롯 추가

실제 점이 그래프 양 끝에 붙지 않도록 labels와 data 앞뒤에 빈 값을 추가합니다.

2

최신 4회만 사용

응시 이력이 5회 이상이면 전체 데이터 중 최신 4개만 잘라 PDF 리포트에 사용합니다.

3

상단 문구로 해석 보완

그래프만으로 의미 전달이 어려우므로 직전 대비 점수와 등급 변화를 상단에 표시합니다.

/* 1회 응시 */ labels: ["", "26.3", ""] data: [null, 450, null] title: "첫 평가 기준점" /* 2회 응시 */ labels: ["", "24.9", "25.2", ""] data: [null, 670, 675, null] title: "직전 대비 +5점 · 등급 유지" /* 3회 응시 */ labels: ["", "24.9", "25.2", "25.3", ""] data: [null, 670, 675, 795, null] title: "직전 대비 +120점 · 5등급 → 4등급" /* 4회 이상 응시: PDF 리포트에는 최신 4회만 표시 */ labels: ["", "24.3", "24.9", "25.2", "25.3", ""] data: [null, 640, 670, 675, 795, null] title: "최근 4회 기준 · 직전 대비 +120점"

5. 그래프 표시 옵션 제안

항목 권장값 이유
최대 표시 회차 최근 4회 최근 2년 흐름을 보여주면서도 PDF 카드 안에서 라벨 가독성을 유지할 수 있음
최소 표시 방식 1회는 중앙 점 점 하나가 좌측 또는 하단에 치우치면 오류처럼 보일 수 있음
점 크기 과거 5~6px, 최신 7~8px 최신 결과가 가장 중요하므로 현재 점을 시각적으로 강조
선 굵기 2~3px PDF 출력 시 선이 너무 두꺼우면 작은 변화가 과장되어 보임
y축 숫자 숨김 또는 최소화 작은 카드에서는 축보다 점수 라벨과 변화량 문구가 더 중요함
데이터 라벨 점수 + 등급 학부모는 점수뿐 아니라 등급 변화까지 함께 봐야 결과를 이해하기 쉬움
5회 이상 이력 최근 4회만 표시 + 안내 문구 “최근 4회 기준”이라는 문구를 넣어 전체 이력 중 일부임을 명확히 함

최종 정리

꺾은선 그래프 하나만 사용해야 한다면, 그래프 종류를 바꾸는 대신 데이터 배치 규칙을 조정해야 합니다. 1회는 중앙 기준점, 2회는 직전 대비 비교, 3회 이상은 추이로 해석하고, 모든 경우 앞뒤 빈 슬롯을 추가해 점이 양 끝에 붙지 않게 하는 것이 핵심입니다.

최대 표시 회차는 최근 4회를 권장합니다. 4회는 6개월 단위 평가 기준으로 최근 2년의 흐름을 보여주며, PDF 리포트의 작은 카드 안에서도 점수·등급·날짜 라벨을 비교적 안정적으로 표시할 수 있는 현실적인 상한입니다.
