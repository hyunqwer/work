현장관리 이력 시스템 — 개설·해지·후속조치를 한 곳에서

서비스 주소: https://yoons-field-note.web.app
Service Overview
필드노트란?

지역본부장과 본사 매니저가 센터 현장 활동, 개설·해지 진행상황, 후속조치 이행을 실시간으로 공유·추적하는 내부 업무 시스템입니다.

📝
이력 등록
상담·민원·면담·해지 등 현장 활동을 바로 기록
🔁
후속조치 추적
미처리→진행중→완료 상태로 이슈 진행상황 관리
🏗️
개설·해지 현황
개설 8단계 / 해지 8단계 체크리스트로 진행 추적
📊
AI 주간 리포트
GPT가 주요 이슈를 요약한 주간·월간 리포트 자동 생성
Key Features
주요 기능

현장 → 본사 정보 흐름을 하나의 화면에서 관리합니다.

🏢

센터별 활동 타임라인

센터 단위로 모든 이력을 시간순으로 조회. 이슈 유형·상태 필터로 빠르게 탐색.

👤

지역본부장별 활동 현황

본부장별 담당 센터 전체 이력 확인. 본부장은 로그인 시 본인 활동만 조회.

✅

후속조치 일괄 관리

후속조치가 있는 이슈만 모아 상태 변경(미처리→완료). 담당자 간 진행상황 공유.

🏗️

개설·해지 파이프라인

단계별 체크리스트 + 소통 스레드. 본사와 현장이 같은 화면에서 진행상황 확인.

🔒

내부메모 권한 분리

본사 전용 내부메모 기능. 지역본부장에게는 자동으로 숨김 처리.

🤖

AI 요약 리포트

GPT-4o-mini가 기간별 이슈를 분석해 주간·월간 리포트 초안 자동 생성.

Roles
사용자 역할 정의

3가지 역할로 접근 권한이 구분됩니다.

⚙️ 관리자

Admin

팀장·이사급 / 시스템 전체 관리

  • 모든 페이지 전체 접근
  • 사용자 계정·역할 생성 및 관리
  • 센터·지역본부장 등록
  • 내부메모 작성 및 열람
  • 개설·해지 파이프라인 생성
  • AI 리포트 생성
💼 매니저

Manager

본사 담당 매니저

  • 전체 센터·본부장 이력 조회
  • 이력 등록 (내부메모 포함)
  • 후속조치 상태 관리
  • 개설·해지 파이프라인 생성·관리
  • AI 주간·월간 리포트 조회
  • 관리자 설정 접근 불가
👤 지역본부장

Director

현장 지역본부장

  • 본인 담당 센터만 조회
  • 담당 센터에 이력 등록
  • 내부메모 열람 불가
  • 본인 후속조치 확인·상태 변경
  • 개설·해지 현황 확인 + 소통
  • 리포트·관리자 설정 접근 불가
