01. SUMMARY

회의 핵심 요지

회의 전체를 관통하는 핵심 방향만 압축 정리했습니다.

핵심 방향

중등 디지털은 ‘교재 전환’이 아니라 ‘서비스 재설계’

지류 교재를 화면으로 옮기는 수준이 아니라, 학습 프로세스·리포트·교사 관리까지 포함한 중등 학습 서비스로 설계 필요.

우선 개발

1차 개발축은 문법과 리딩

현장 수요와 개발 가능성을 고려해 문법·리딩을 먼저 구체화하고, 구문·듣기·어휘는 후속 모듈로 확장 검토.

검증 방식

프로토타입과 파일럿 중심 접근

올해는 완성 상품보다 프로토타입 제작과 현장 검증에 초점을 두고, 내년 하반기~내년 말 정식 오픈 가능성 검토.

02. DECISIONS

결정사항 및 잠정 합의 방향

회의 중 방향성이 비교적 명확하게 정리된 사항입니다. 세부 실행안은 후속 논의에서 조정 가능합니다.

결정 1

개발 우선순위

  • 중등 디지털 1차 개발 영역은 문법과 리딩 중심으로 설정
  • 전체 라인업을 한 번에 개발하지 않고, 대표 영역부터 프로토타입 제작
  • 문법·리딩을 통해 디지털 학습 프로세스의 기본 모델 검증
  • 구문·듣기·어휘·쓰기는 2차 확장 영역으로 분류
결정 2

문법 라인

  • 문법은 현장 수요가 높고 디지털화 효과가 큰 영역으로 판단
  • 개념 학습형 문법과 문제 훈련형 문법의 역할 구분 필요
  • 디지털에서는 충분한 문제량, 자동 채점, 오답 관리, 반복 훈련 제공 가능
  • 학년별 문법 범위를 기준으로 중1·중2·중3 라인 구성 검토
결정 3

리딩 라인

  • 리딩은 기존 윤선생의 영어 입력 중심 강점을 이어가는 핵심 라인
  • 기본 리딩은 ESL 리딩 성격을 유지하되, 중등 독해력·문해력 요소 보강
  • 단순 지문+문제풀이가 아니라 디지털 학습 흐름으로 설계
  • 프리리딩, 영상, 본문 기반 단어, 구문 설명, 낭독, 리포트까지 포함 검토
결정 4

구문 독해 접근

  • 구문 독해 니즈는 인정하되, 독립 라인 개발은 즉시 확정하지 않음
  • 우선 리딩 본문 안에서 핵심 문장 중심 구문 훈련 모듈로 검증
  • 천일문식 독립 구문과 리딩 본문 기반 구문은 성격을 구분
  • 현장 반응 확인 후 별도 구문 라인화 여부 판단
결정 5

듣기·어휘 방향

  • 듣기는 단순 모의고사 교재보다 문제은행형·실전 대비 모듈로 검토
  • 어휘는 별도 교재보다 개인화 디지털 어휘장·누적 복습 모듈이 적합
  • 수능 빈출 단어, 어원 단어 등은 선택형 부가 콘텐츠로 확장 가능
  • 두 영역 모두 1차 핵심 개발보다는 후속 모듈 성격으로 검토
결정 6

워크북·종이 자료

  • 디지털 중심 전환을 목표로 하되 종이 자료 필요성은 열어둠
  • 초기에는 대규모 워크북 제작보다 PDF·출력형 보조 자료 가능성 검토
  • 현장 니즈가 확인되면 워크북 상품화 가능
  • 중간 단계에서는 디지털+워크북의 하이브리드 운영 가능성 높음
03. READING PROTOTYPE

리딩 프로토타입 핵심 설계 방향

리딩 파워 콘텐츠를 활용한 디지털 리딩 프로토타입 검토 내용을 구조화했습니다.

1
도입 — 제목, 프리리딩 질문, 시각 자료 또는 짧은 영상으로 주제 맥락 형성
2
본문 기반 단어 학습 — 단어 리스트 분리 제시보다 본문 안에서 단어를 클릭하고 문맥 속 의미 유추
3
본문 읽기 — 글의 전체 흐름을 먼저 파악한 뒤, 필요한 부분을 정밀 학습
4
구문 설명 — 모든 문장이 아니라 핵심 문장 중심으로 짧은 설명과 이해 확인 활동 제공
5
낭독·발화 활동 — 단락 또는 핵심 문장 낭독을 통해 소리 학습 강점 유지
6
리포트 — 학습 완료율, 이해도, 단어, 구문, 낭독 결과를 교사·학부모 피드백으로 연결
설계 포인트: 기존 SB처럼 긴 강의를 강제로 듣게 하는 방식은 지루함이 반복될 수 있음. 짧은 해설, 클릭형 상호작용, 중간 확인 문제, 낭독 피드백을 조합해 디지털 리딩만의 학습 경험을 만드는 방향이 적합함.
04. OPEN ISSUES

추가 논의할 사항

아직 결정되지 않았거나, 현장 검증 후 판단해야 할 쟁점입니다.

논의 1

전체 학제 구조

  • 학년별 라인으로 갈지, 수준별 라인으로 갈지 결정 필요
  • 기본 리딩, 문제풀이형 독해, 수능형 독해의 포지션 구분 필요
  • 중등 라인과 고등·수능 라인의 연결 구조 검토 필요
논의 2

문법 세부 구성

  • 개념형과 문제훈련형을 분리할지, 하나의 흐름으로 통합할지 결정 필요
  • 한 레벨당 적정 차시 수 확정 필요
  • 내신형 문제와 기본 훈련 문제의 비중 설계 필요
  • 오답 반복, 누적 복습, 리포트 제공 방식 구체화 필요
논의 3

리딩 학습 흐름

  • 한 차시의 적정 학습 시간 설정 필요
  • 구문 설명을 필수로 들려줄지, 선택형으로 제공할지 검토 필요
  • 낭독 활동을 필수 활동으로 둘지, 선택 활동으로 둘지 검토 필요
  • 영상·단어·본문·문제풀이의 최적 순서 검증 필요
논의 4

구문 라인화 여부

  • 현장이 원하는 구문이 천일문식 독립 구문인지 확인 필요
  • 리딩 지문 기반 구문 훈련으로 충분한지 검증 필요
  • 문법 스코프와 구문 스코프를 어떻게 구분할지 정리 필요
논의 5

파일럿 설계

  • 1차시 체험형 파일럿과 1개월 사용형 파일럿 중 운영 방식 결정 필요
  • 의미 있는 파일럿을 위한 최소 차시 수 검토 필요
  • 현장 영향력이 큰 교사·원장 중심의 참여자 선정 필요
  • 학습 몰입도, 교사 운영 편의성, 가격 수용성 측정 지표 필요
논의 6

가격·상품 패키지

  • 월 구독료 수준과 상품 가치 구성 검토 필요
  • 문법·리딩 단독 판매와 중등 디지털 패키지 판매 비교 필요
  • 리포트, 교사용 관리 화면, 워크북 포함 범위 결정 필요
  • 구독형 BM을 지탱할 회원 수 규모 검토 필요

07. REVIEW

작성자 검토 의견

핵심 판단

  • 이번 회의의 핵심 결론은 문법과 리딩을 중심으로 중등 디지털의 표준 학습 경험을 먼저 만든다는 점임.
  • 구문은 매력적인 영역이지만, 독립 라인으로 즉시 개발하기보다 리딩 안에서 먼저 검증하는 접근이 현실적임.
  • 듣기와 어휘는 별도 교재보다 문제은행·개인화 복습·부가 모듈 형태가 디지털 상품에 더 적합함.
  • 성공 여부는 콘텐츠 자체뿐 아니라 교사용 관리, 학부모 리포트, 워크북 보완, 가격 수용성까지 함께 설계하는 데 달려 있음.
  • 파일럿은 단순 테스트가 아니라 현장 확산의 출발점이므로, 영향력 있는 현장 그룹을 전략적으로 선정할 필요가 있음.
